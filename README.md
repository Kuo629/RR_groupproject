## Course

## Proposed Topic
Credit Risk Analysis

## Project Overview

This project aims to reproduce and extend an existing credit risk analysis workflow originally implemented in R by translating it into Python. The goal is to ensure full reproducibility while comparing analytical approaches and results across programming languages.

## Source Repository (R Implementation)

The original R-based project is available here:
https://github.com/KevRB3/Deuces/tree/main

This repository will serve as the baseline for:

* Code translation from R to Python
* Validation and comparison of results
* Understanding the original analytical workflow

## Candidate Dataset
Loan Approval Classification Data (Kaggle)  
https://www.kaggle.com/datasets/taweilo/loan-approval-classification-data

## Project Objectives
* Reproduce the original R analysis in Python
* Ensure computational reproducibility
* Compare results between R and Python implementations
* Apply reproducible research practices, including:
 * Version control
 * Environment management
 * Clear and complete documentation
Extend the analysis with additional models or evaluation methods if appropriate

## Methodology
# Review and document the original R workflow
# Recreate data preprocessing steps in Python
# Implement equivalent machine learning models
# Validate outputs against the original results
